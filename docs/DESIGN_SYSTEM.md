# Danny Road 设计规范

## 设计原则

### 1. 信息三级分层

**P0 核心信息**（红色高亮）：
- 停车便利度
- 停车费用
- 车位数量

**P1 路线信息**（正常显示）：
- 景点名称
- 餐厅推荐
- 距离/时长

**P2 辅助信息**（灰色小字）：
- 厕所位置
- 充电站类型
- 用户评分

---

### 2. 色彩系统

| 类型 | 颜色 | 用途 |
|-----|------|------|
| **停车红** | `#FF4444` | 停车信息、警告 |
| **充电蓝** | `#0066FF` | 充电站、科技 |
| **厕所橙** | `#FF9900` | 厕所、服务 |
| **成功绿** | `#00CC66` | 可用/空闲 |
| **警告黄** | `#FFCC00` | 紧张/有限 |
| **背景白** | `#FFFFFF` | 卡片背景 |
| **背景灰** | `#F5F5F5` | 页面背景 |

---

### 3. 字体系统

| 类型 | 大小 | 字重 | 用途 |
|-----|------|------|------|
| **大标题** | 32px | Bold | 页面标题 |
| **标题** | 24px | Bold | 模块标题 |
| **副标题** | 20px | Medium | 卡片标题 |
| **正文** | 16px | Regular | 主要内容 |
| **辅助** | 14px | Regular | 次要信息 |
| **提示** | 12px | Regular | 说明文字 |

---

### 4. 间距系统

| 类型 | 大小 | 用途 |
|-----|------|------|
| **极小** | 4px | 图标间距 |
| **小** | 8px | 元素间距 |
| **中** | 16px | 卡片内边距 |
| **大** | 24px | 模块间距 |
| **极大** | 32px | 页面边距 |

## 核心组件

### 1. ServiceCard (服务设施卡片)

```tsx
<div className="service-card parking-highlight">
  <div className="icon">🅿️</div>
  <div className="content">
    <h4>停车场</h4>
    <p className="parking-info">
      <span className="status available">充足</span>
      <span className="price">¥10/小时</span>
    </p>
    <p className="distance">距离 200m</p>
  </div>
</div>
```

**设计要点**：
- 停车信息用红色边框
- 状态用颜色区分（绿=充足/黄=紧张/红=已满）
- 价格/距离醒目展示

## 交互规范

### 点击反馈
- 按钮点击：缩放 0.95 + 阴影加深
- 卡片点击：边框高亮 + 轻微上浮

### 加载状态
- 骨架屏（灰色占位）
- 加载动画（旋转图标）

### 空状态
- 插图 + 提示文字
- 引导操作按钮

### 错误处理
- 错误提示（红色背景）
- 重试按钮
